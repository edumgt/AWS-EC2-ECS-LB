# AWS EC2 Auto - 0702 Spring Boot 준비

## 개요
- 기존 AMI에 Spring Boot를 올리기 위한 JDK/FTP 설정 절차를 정리했습니다.
- 명령어 중심으로 기록되어 있어 단계별로 실행하면 됩니다.
- 보안상 민감할 수 있는 IP/계정 정보는 마스킹 처리했습니다.

## AMI에 Spring Boot 올리기 위한 작업
기존 AMI 중 Nginx 미설치 AMI 인스턴스를 생성합니다.
```bash
sudo apt update
sudo apt install openjdk-17-jdk
```
```bash
root@<PRIVATE_IP>:~# java --version
openjdk 17.0.15 2025-04-15
OpenJDK Runtime Environment (build 17.0.15+6-Ubuntu-0ubuntu122.04)
OpenJDK 64-Bit Server VM (build 17.0.15+6-Ubuntu-0ubuntu122.04, mixed mode, sharing)
```

## FTP 데몬 설치
```bash
sudo apt install vsftpd -y
sudo systemctl start vsftpd
sudo systemctl enable vsftpd
vi /etc/vsftpd.conf
```

### vsftpd 설정(쓰기 권한)
```ini
anonymous_enable=NO
local_enable=YES
write_enable=YES
```

```bash
sudo systemctl restart vsftpd
```

## 사용자 생성
```bash
sudo useradd -m ftpuser
sudo passwd ftpuser
```
패스워드: `<REDACTED_PASSWORD>`

## 앱 실행
```bash
java -jar ./app.jar
```
