# EC2 실습 08 - 운영 점검 명령 모음

아래 명령은 장애 대응 시 빠른 확인을 위해 정리한 조회 템플릿입니다.

## ALB가 사용 중인 Subnet 확인
```bash
aws elbv2 describe-load-balancers \
  --names <LB_NAME> \
  --query "LoadBalancers[].AvailabilityZones[].SubnetId"
```

## Subnet과 AZ 매핑 확인
```bash
aws ec2 describe-subnets \
  --subnet-ids subnet-xxxxxxxx subnet-yyyyyyyy \
  --query "Subnets[].{SubnetId:SubnetId,AZ:AvailabilityZone,VpcId:VpcId}" \
  --output table
```

## Subnet에 연결된 라우팅 경로 확인
```bash
aws ec2 describe-route-tables \
  --filters Name=association.subnet-id,Values=subnet-xxxxxxxx,subnet-yyyyyyyy \
  --query "RouteTables[].Routes[]"
```

## 보안그룹 상세 확인
```bash
aws ec2 describe-security-groups \
  --group-ids sg-xxxxxxxx \
  --query "SecurityGroups[0].{GroupName:GroupName,VpcId:VpcId,Inbound:IpPermissions,Outbound:IpPermissionsEgress}" \
  --output json
```

## Target Group 상태 확인
```bash
aws elbv2 describe-target-health \
  --target-group-arn <TARGET_GROUP_ARN>
```
