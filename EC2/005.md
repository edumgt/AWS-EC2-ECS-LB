# EC2 실습 05 - 애플리케이션 런타임 준비

## 목표
- 인스턴스에 Java 런타임과 파일 전송 환경을 준비합니다.
- 운영 관점에서 최소 보안 설정을 적용합니다.

## 1) JDK 설치
```bash
sudo apt update
sudo apt install -y openjdk-17-jdk
java --version
```

## 2) 파일 전송(권장: SFTP)
`vsftpd` 대신 SSH 기반 SFTP를 우선 사용합니다.

예시:
```bash
sftp -i <KEY_NAME>.pem ubuntu@<PUBLIC_IP>
```

## 3) (선택) FTP 데몬 사용 시 최소 설정
```bash
sudo apt install -y vsftpd
sudo systemctl enable --now vsftpd
```

`/etc/vsftpd.conf` 핵심:
```ini
anonymous_enable=NO
local_enable=YES
write_enable=YES
```

변경 후:
```bash
sudo systemctl restart vsftpd
```

## 4) 앱 실행 예시
```bash
java -jar app.jar
```

## 보안 메모
- 패스워드를 문서에 평문으로 기록하지 않습니다.
- 가능하면 IAM Role + SSM Session Manager로 운영 접근을 대체합니다.
